Created for (and currently standing in) the Central Library Cologne by a team of two Ph.D. students as well as four student assistants. Quellentaucher is a research prototype for the "LibrOS" project.

The application fetches news articles from a local news-website and enriches these articles with additional information from the web, such as Tweets, Wikipedia articles or relevant books from the local library. The user is then able to select a book in a virtual bookshelf to gain more information about this book.

My job was to implement a big part of the article, keyword and information generation as well as translated most design ideas into an actual User Interface.

MediaRoomControl

Programming languages

C, C#

Timeframe

Oct 2013 - Feb 2014

Developed by a team of three students for the course of "Blended Interaction" at the University of Konstanz.

MediaRoomControl uses spatial data to let one keyboard send its input over a network to other displays. SSelecting a display happens with by touching the display with a keyboard.

The project uses TCP for input transfer and communication via remote procedure calls between clients and the server, as well as UDP for IPv4 address discovery and automatic connecting. The input is intercepted on a RaspberryPi / Debian system, sent to a server running the OptiTrack and Proximity Toolkit for spatial data processing.

This research project is currently standing in the library at the University of Konstanz, but will also be installed later this year in the Central Library Cologne.

Facet Search++ enables a group of users to simultaneously search for books; in the result view they are able to compile a list of books, show their location in the library and print out additional information to help them search the books.
